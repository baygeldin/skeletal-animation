Запускать нужно на сервере, потому что используются HTTP запросы.
(например, установить Node.js, набрать в терминале "npm install http-server -g",
зайти в папку с программой и набрать в терминале "http-server", после чего
зайти в браузер на localhost:8080, или другой порт, который будет написан в терминале)

Поворачивать модель можно стркелками, запустить/остановить скелетную анимацию можно по пробелу.
